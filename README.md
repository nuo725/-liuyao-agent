# Liuyao Agent Project

This workspace is organized by responsibility:

- `frontend/`: Flutter app frontend.
- `backend/`: backend service scaffold and server-side implementation.
- `docs/`: classified documentation for frontend, backend, and project requirements.
- `六爻/`: Liuyao knowledge base materials.

## Current Entry Points

- Frontend app: `frontend/README.md`
- Backend service: `backend/README.md`
- Documentation index: `docs/README.md`

## Local Backend

The backend scaffold listens on:

```text
http://127.0.0.1:3000/api/v1
```

Android emulator builds should use:

```text
http://10.0.2.2:3000/api/v1
```
